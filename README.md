# 💻 Bilal Zakaria — Portfolio Website

This is my personal portfolio website, built to showcase my projects, technical skills, and professional background.  
It highlights my work in software engineering, data systems, and full-stack development, with a clean, modern design built using **React**, **Vite**, and **Tailwind CSS**.

---

## 🚀 Tech Stack

- **Frontend:** React (Vite), Tailwind CSS  
- **Hosting:** Vercel  
- **Version Control:** Git + GitHub  
- **Design:** Responsive, mobile-first layout using modern UI patterns

---

## 🧩 Features

- 📱 **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- 🧭 **Smooth Navigation:** Animated, sticky navbar with section scrolling.
- 💡 **Dynamic Sections:** Hero, About, Projects, Resume, and Contact.
- ⚡ **Fast Build:** Powered by Vite for blazing-fast development and build times.
- 🪶 **Minimalist UI:** Inspired by modern developer portfolios, with typography and color balance for readability.
- 🔗 **Project Highlights:** Links to live demos, GitHub repositories, and detailed project summaries.
